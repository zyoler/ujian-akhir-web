About Me

Hi, saya adalah calon programer yang berdomisili di Bandung. Saat ini saya sedang berkuliah di Universitas Nasional PASIM dalam jalur PUB. Program studi yang saya ambil adalah D3-Manajemen Informatika, saya belajar pemrograman dengan mengikuti pelatihan yang disediakan oleh program PUB ini dan untuk tambahannya saya cari cari materi dari media internet. Saya memang termasuk baru dalam dunia pemograman ini, sebab itu saya akan lebih rajin lagi dalam belajar pemograman ini agar saya bisa mengusai dengan baik bahasa-bahasa pemograman.
